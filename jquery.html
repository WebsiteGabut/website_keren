Tutorial jQuery

Pelajari dasar-dasar jQuery untuk mempermudah manipulasi DOM dan interaktivitas web.

Apa itu jQuery?

jQuery adalah pustaka JavaScript yang dirancang untuk menyederhanakan manipulasi HTML, penanganan acara, animasi, dan interaksi dengan AJAX. jQuery memudahkan pengembang web untuk menulis kode yang lebih sedikit dan lebih efisien.

Menambahkan jQuery ke Halaman Web

Anda dapat menambahkan jQuery ke halaman web Anda dengan cara berikut:


<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

Dasar-Dasar jQuery

Berikut adalah beberapa konsep dasar dalam jQuery:

  • Memilih Elemen: Menggunakan selector jQuery untuk memilih elemen HTML.
    • 
$(document).ready(function() {
    $("h1").css("color", "blue");
});
  • Menangani Acara: Menggunakan jQuery untuk menangani berbagai jenis acara seperti klik, hover, dll.
    • 
$("#myButton").click(function() {
    alert("Tombol diklik!");
});
  • Manipulasi DOM: Menambahkan, menghapus, atau mengubah elemen HTML.
    • 
$("#myDiv").html("<p>Konten baru</p>");
$("#myDiv").append("<p>Konten tambahan</p>");
  • Animasi: Menggunakan jQuery untuk membuat efek animasi.
    • 
$("#myDiv").fadeIn();
$("#myDiv").slideUp();
  • AJAX: Menggunakan jQuery untuk melakukan permintaan AJAX.
    • 
$.ajax({
    url: "data.json",
    dataType: "json",
    success: function(data) {
        console.log(data);
    }
});

Contoh Penggunaan

Berikut adalah contoh penggunaan jQuery untuk membuat halaman web interaktif:


<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Contoh jQuery</title>
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <script>
        $(document).ready(function() {
            $("#myButton").click(function() {
                $("#myDiv").fadeToggle();
            });
        });
    </script>
</head>
<body>
    <h1>Selamat Datang</h1>
    <button id="myButton">Tampilkan/Sembunyikan</button>
    <div id="myDiv" style="display:none;">
        <p>Ini adalah konten yang dapat ditampilkan atau disembunyikan.</p>
    </div>
</body>
</html>
