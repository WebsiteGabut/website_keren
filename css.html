Tutorial CSS

Pelajari dasar-dasar CSS untuk mempercantik halaman web Anda.

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web. CSS memungkinkan Anda untuk mengubah warna, font, layout, dan banyak aspek lainnya dari elemen HTML.

Struktur Dasar CSS


/* Ini adalah komentar di CSS */
selector {
    property: value;
}

Metode Penerapan CSS

  • Inline CSS: Menggunakan atribut style langsung pada elemen HTML.
    • 
<p style="color: red;">Ini adalah paragraf dengan teks merah.</p>
  • Internal CSS: Menyisipkan CSS di dalam tag <style> di bagian <head> dari dokumen HTML.
    • 
<style>
    body {
        background-color: lightblue;
    }
</style>
  • External CSS: Menggunakan file CSS terpisah yang di-link ke dokumen HTML.
    • 
<link rel="stylesheet" href="style.css">

Selector CSS

Selector digunakan untuk memilih elemen yang akan diberikan gaya. Berikut adalah beberapa jenis selector:

  • Selector Tag: Memilih semua elemen dari tipe tertentu.
    • 
h1 {
    color: blue;
}
  • Selector Kelas: Memilih elemen dengan kelas tertentu (didefinisikan dengan .).
    • 
.button {
    background-color: green;
    color: white;
}
  • Selector ID: Memilih elemen dengan ID tertentu (didefinisikan dengan #).
    • 
#header {
    font-size: 24px;
}

Properti CSS Umum

  • color: Mengatur warna teks.
  • background-color: Mengatur warna latar belakang.
  • font-size: Mengatur ukuran font.
  • margin: Mengatur ruang di luar elemen.
  • padding: Mengatur ruang di dalam elemen.
  • border: Mengatur garis batas elemen.

Contoh Penggunaan

Berikut adalah contoh penggunaan CSS untuk mendesain halaman HTML:


<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Contoh CSS</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1 class="heading">Selamat Datang</h1>
    <p class="description">Ini adalah contoh halaman dengan CSS.</p>
    <button class="button">Klik Saya</button>
</body>
