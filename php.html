Tutorial PHP

Pelajari dasar-dasar PHP untuk pengembangan web dinamis.

Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang dirancang untuk pengembangan web. PHP memungkinkan Anda untuk membuat halaman web yang dinamis dan interaktif dengan memproses data dari server.

Menambahkan PHP ke Halaman Web

PHP dapat ditambahkan ke halaman web dengan cara berikut:


<?php
    echo "Hello, World!";
?>

File PHP biasanya memiliki ekstensi .php dan dapat disisipkan di dalam file HTML.

Dasar-Dasar PHP

Berikut adalah beberapa konsep dasar dalam PHP:

  • Variabel: Tempat untuk menyimpan data.
    • 
$nama = "John";
$umur = 25;
  • Tipe Data: PHP mendukung berbagai tipe data, termasuk string, integer, float, boolean, dan array.
    • 
$isActive = true; // boolean
$jumlah = 100; // integer
$harga = 99.99; // float
$warna = array("merah", "hijau", "biru"); // array
  • Fungsi: Kumpulan kode yang dapat dipanggil untuk melakukan tugas tertentu.
    • 
function sayHello() {
    return "Hello, World!";
}
echo sayHello();
  • Pengkondisian: Menggunakan pernyataan if untuk membuat keputusan dalam kode.
    • 
if ($umur >= 18) {
    echo 'Anda sudah dewasa.';
} else {
    echo 'Anda masih anak-anak.';
}
  • Perulangan: Menggunakan pernyataan for atau while untuk mengulangi blok kode.
    • 
for ($i = 0; $i < 5; $i++) {
    echo $i;
}

Contoh Penggunaan

Berikut adalah contoh penggunaan PHP untuk membuat halaman web dinamis:


<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Contoh PHP</title>
</head>
<body>
    <h1>Selamat Datang</h1>
    <?php
        echo "<p>Ini adalah contoh penggunaan PHP.</p>";
    ?>
</body>
</html>

Kesimpulan

PHP adalah bahasa pemrograman yang sangat populer untuk pengembangan web. Dengan memahami dasar-dasar PHP, Anda dapat membuat aplikasi web yang dinamis dan interaktif. PHP memungkinkan Anda untuk berinteraksi dengan database, mengelola sesi pengguna, dan banyak lagi.
