Tutorial JavaScript

Pelajari dasar-dasar JavaScript untuk menambahkan interaktivitas pada halaman web Anda.

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. Dengan JavaScript, Anda dapat mengubah konten, mengontrol multimedia, membuat animasi, dan banyak lagi.

Menambahkan JavaScript ke Halaman Web

JavaScript dapat ditambahkan ke halaman web dengan cara berikut:

  • Inline JavaScript: Menyisipkan kode JavaScript langsung ke dalam atribut HTML.
    • 
<button onclick="alert('Hello, World!')">Klik Saya</button>
  • Internal JavaScript: Menyisipkan kode JavaScript di dalam tag <script> di bagian <head> atau <body> dari dokumen HTML.
    • 
<script>
    alert('Hello, World!');
</script>
  • External JavaScript: Menggunakan file JavaScript terpisah yang di-link ke dokumen HTML.
    • 
<script src="script.js"></script>

Dasar-Dasar JavaScript

Berikut adalah beberapa konsep dasar dalam JavaScript:

  • Variabel: Tempat untuk menyimpan data.
    • 
var nama = "John";
let umur = 25;
const negara = "Indonesia";
  • Tipe Data: JavaScript memiliki beberapa tipe data, termasuk string, number, boolean, dan objek.
    • 
let isActive = true; // boolean
let jumlah = 100; // number
let nama = "Jane"; // string
  • Fungsi: Kumpulan kode yang dapat dipanggil untuk melakukan tugas tertentu.
    • 
function sayHello() {
    alert('Hello, World!');
}

    Memanggil fungsi:

    
sayHello();
  • Pengkondisian: Menggunakan pernyataan if untuk membuat keputusan dalam kode.
    • 
if (umur >= 18) {
    alert('Anda sudah dewasa.');
} else {
    alert('Anda masih anak-anak.');
}
  • Perulangan: Menggunakan pernyataan for atau while untuk mengulangi blok kode.
    • 
for (let i = 0; i < 5; i++) {
    console.log(i);
}
